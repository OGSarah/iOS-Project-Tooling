# Scripts, SwiftLint, and other universal files

- ### For adding the SarahUniverse copyright to existing files in an Xcode project:
  Add_Swift_Header.sh

  From the main project folder copy and paste into the terminal (if you're using bash in the terminal):
  
  ```bash add_swift_header.sh . --project ProjectName --git-year --apply```

- ### Script to add to the Build Phases in Xcode for SwiftLint:
  RunScript.sh (don't run it, just copy and past into the shell inside Run Script in Build Phases)

- ### Basic SwiftLint file for a new Xcode project:
  .swiftlint.yml

- ### Agents.md file for mostly SwiftUI but some UIKit and Swift only Xcode projects
  Agents.md
